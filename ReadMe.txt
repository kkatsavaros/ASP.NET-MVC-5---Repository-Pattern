ASP.NET MVC 5 - Repository Pattern


This example shows the creation of a CRUD application using Repository Pattern.



Database: RepositoryPattern
Table: Employees (Id,Names,Email,Salary)

ADO.NET Entity Data Model: EmployeeModel

Connection name in Web.Config: RepositoryPatternEntities

ModelNamespace: RepositoryPatternModel

Interface: Iemployee.cs
Implemantation του Interface class: repositoryEmployee.cs

Controller:EmpController
